# DejaVu 输出协议

这份文档定义 `DejaVu` 负责输出的显示协议。这里只描述“要画成什么”, 不讨论 Python 端怎么解码实现。

## 术语

- `Matrix`: 像素矩阵, 当前实现是按 Cell 网格排布
- `Cell`: 4x4 像素块, 主要表达布尔和数值映射
- `MegaCell`: 8x8 纯色块
- `BadgeCell`: 8x8 图标块, 右下角附带 2x2 脚标
- `CharCell`: 8x8 文本像素块

## 坐标

- 矩阵坐标以 Cell 为基本单位
- 当前实现里的 Matrix 大小是 `84 x 28`
- 这个尺寸是当前实现, 不应该被当成永远不变的真理

## 输出约束

- `DejaVu` 输出的是显示协议, 不是 WoW 内部原始状态直通
- 颜色、图标类型、标记区, 都是协议字段, 不只是“看起来好看”
- 如果协议要改, 先改这份文档, 再安排代码变更

## 协议内容

- Cell 类型和尺寸语义
- Matrix 坐标与尺寸语义
- 颜色与标记区语义
- 哪些区域是稳定锚点, 哪些区域是动态内容

颜色细节单独见 `03_color_conventions.md`。
